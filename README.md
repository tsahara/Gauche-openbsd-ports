Gauche-openbsd-ports
====================

OpenBSD port of Gauche a scheme implementation.

## How to use
```
% cd /usr/ports/lang
% git clone https://github.com/tsahara/Gauche-openbsd-ports.git gauche
% cd gauche
% make install
```
